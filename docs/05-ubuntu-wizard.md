Status: migration source

## Purpose
Ubuntu exists only to run a local Wizard Server.

## Deployment Modes
### USB-hosted
- Runs from Sonic Stick
- No local persistence

### Local disk (optional)
- Minimal install
- No GUI
- Service-only

## Responsibilities
- Install orchestration
- Validation checks
- Recovery workflows
- API for Tauri UI
