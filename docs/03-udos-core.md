# uDOS Core (Alpine Linux)

Status: migration source
Primary learning destination: `courses/02-system-provisioning/lessons/01-runtime-roles.md`

## Base System
- Alpine Linux
- musl libc
- BusyBox
- openrc init

## Interface
- TUI-first (ncurses)
- Optional Tauri thin GUI
- Single fullscreen process
- No window manager

## Responsibilities
- Hardware discovery
- Disk partitioning
- OS installation
- Bootloader management
- Reboot routing
