This module represents the Sonic-owned USB planning and execution lane.

- `apps/sonic-cli/cli.py`
- `services/planner.py`
- `services/manifest.py`
- `services/runtime_service.py`
- `scripts/partition-layout.sh`
- `scripts/apply-payloads-v2.sh`

Educational role:

- teach safe planning and dry-run workflows
