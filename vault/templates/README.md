# Vault Templates

Reusable Markdown templates for Sonic examples belong here.

Use this folder for:

- deployment-note templates
- device-profile templates
- manifest review templates

## Current Contents

- `device-profiles/README.md`
- `device-profiles/*.profile.json` (reference profile examples)

## Usage Pattern

1. Copy a template profile to your working area
2. Adjust partition and boot target assumptions
3. Generate and inspect a dry-run manifest
4. Record outcomes in `vault/deployment-notes/`
